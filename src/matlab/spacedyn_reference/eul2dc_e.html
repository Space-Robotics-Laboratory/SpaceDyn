eul2dc

Grammar

dc = eul2dc( euler )
dc = eul2dc( phi, theta, psi )

Global variables

---

Explanation

Function which calculates a direction cosine matrix from 3-1-3 Euler angles.


Details

Direction Cosine Matrix in SpaceDyn
Suppose that R is a vector subject to rotation around x-axis of an inertial reference frame by roll [rad], and the resulting vector is S.
If another vector X is expressed as IX in the inertial frame,
There is a following relationship among R, S, and a direction cosine C.
IS = Cx( roll ) * IR

R is rotated around z-axis of the inertial reference frame by phi [rad],
followed by another rotation around x-axis of the rotated reference frame by theta [rad],
followed by another rotation around z-axis of the rotated reference frame by psi [rad],
Let T be the resulting vector.
The direction cosine of this rotation is expressed as follows.
C = Cz( psi ) * Cx( theta ) * Cz( phi )
Hence,
IT = C * IR
Therefore, direction cosine matrices are capable of rotating vectors in the same reference frame.
