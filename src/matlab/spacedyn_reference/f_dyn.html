解説

順動力学の計算．

ロボットシステムの運動方程式は以下のとおりである．
H*xdd + c = F + J^(-1)*Fe
H：システム全体の慣性行列
xdd：ロボット全体の加速度
F：一般化力
J：ヤコビ行列
Fe：手先に働く外力
c：速度依存項

ここで，速度依存項 c を直接求めることは効率が悪い．
ロボットの加速度 xdd と外力 Fe を0とすれば，c は一般化力 F と一致する．
よって，加速度 xdd は以下のように表せる．
xdd = H^(-1)*{ F - F|xdd,Fe=0 + J^(-T)*Fe }
