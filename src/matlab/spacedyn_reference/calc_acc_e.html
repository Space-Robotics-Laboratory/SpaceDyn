calc_acc

Grammar

[ vd, wd ] = calc_acc( A0, AA, w0, ww, vd0, wd0, q, qd, qdd )

Global variables

BB, J_type, Qi,
c0, cc,
num_q, Ez

Explanation

Function which calculates linear and angular acceleration of each joint.
vd, wd are linear acceleration [m/s^2] and angular acceleration [rad/s^2].

Return value consists of the following.
vd = [ 3Å~1 3Å~1 3Å~1 Åc 3Å~1 ]
vd1 vd2 vd3 vdn
wd = [ 3Å~1 3Å~1 3Å~1 Åc 3Å~1 ]
wd1 wd2 wd3 wdn
