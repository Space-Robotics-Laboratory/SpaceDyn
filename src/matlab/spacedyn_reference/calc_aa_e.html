calc_aa

Grammar

AA = calc_aa( A0, q )

Global variables

BB, J_type, Qi
num_q

Explanation

Function which calculates coordinate transform matrix IAi of all joints.
Since joint and link reference frames coincides,this function can be seen as the one which calculates coordinate transform matrix of all the centers of mass of the links.

"Coordinate Transform Matrix"
In SpaceDyn, this means the transform matrix from joint reference frame to inertial reference frame.

I R = I A i * i R
IR ÅF vector expressed in inertial reference frame R
iR ÅF vector expressed in Joint i reference frame R
IAi ÅF coordinate transform matrix of Joint i

AA consists of the following.
AA = [ 3Å~3 3Å~3 3Å~3 Åc 3Å~3 ]
IA1 IA2 IA3 IAn
