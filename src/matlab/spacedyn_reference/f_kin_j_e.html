f_kin_j

Grammar

[ POS_j, ORI_j ] = f_kin_j( RR, AA, q, joints )

Global variables

J_type,
cc,
Ez

Explanation

Function which calculates the position and attitude of all joints specified by "joints."


Suppose that an end point is connected to its base through Joint 1, 2, 4, 5, 7 in sequence
( joints = [ 1 2 4 5 7 ] )
In this case, the return values are as follows.

POS_j1 POS_j2 POS_j4 POS_j5 POS_j7
ORI_j = [ 3Å~3 3Å~3 3Å~3 3Å~3 3Å~3 ]
ORI_j1 ORI_j2 ORI_j4 ORI_j5 ORI_j7
