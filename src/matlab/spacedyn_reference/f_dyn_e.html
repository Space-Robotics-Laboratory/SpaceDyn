f_dyn

Grammar

[ vd0, wd0, qdd ] = f_dyn(R0, A0, v0, w0, q, qd, F0, T0, Fe, Te, tau)

Global variables

BB, S0, SS, SE, J_type, Qi, Qe,
c0, cc, ce,
m0, m, mass, inertia0, inertia,
num_q, Ez, Gravity

Explanation

Function which computes forward dynamics.

Details

The equation of motion of a robot system is shown below.
H*xdd + c = F + J^(-1)*Fe
H：Inertia matrix of the whole robot system
xdd：Acceleration of the whole robot system
F：Generalized forces
J：Jacobian matrix
Fe：External forces applied to an end point
c：Velocity-dependent term

Directly determining the velocity-dependent term c is computationally expensive.
If the acceleration xdd of the robot and the external forces Fe are set to zero,
the velocity-dependent term c becomes equal to the generalized forces F.
Thus, the acceleration xdd of the robot can be expressed as follows.
xdd = H^(-1)*{ F - F|xdd,Fe=0 + J^(-T)*Fe }
