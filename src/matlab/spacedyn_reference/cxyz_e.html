Explanation

Function which calculates a direction cosine matrix describing the rotation around principal axes of rotation.


Details


Direction Cosine Matrix in SpaceDyn
Suppose that R is a vector subject to rotation around x-axis of an inertial reference frame by roll [rad], and the resulting vector is S.
If another vector X is expressed as IX in the inertial frame,
There is a following relationship among R, S, and a direction cosine C.
IS = Cx( roll ) * IR

R is rotated around x-axis of the inertial reference frame by roll [rad],
followed by another rotation around y-axis of the rotated reference frame by pitch [rad],
followed by another rotation around z-axis of the rotated reference frame by yaw [rad],
Let the resulting vector be T.
The direction cosine of this rotation is expressed as follows.
C = Cz( yaw ) * Cy( pitch ) * Cx( roll )
Hence,
IT = C * IR

Therefore, direction cosine matrices are capable of rotating vectors in the same reference frame.
