tilde

Grammar

B = tilde( vector )

Global variables

---

Explanation

Function which calculates a skew-symmetric matrix.

Details

Consider a cross product of two 3-by-1 vectors, r and v.
a = rÅ~v
There exists a square matrix satisfying the following condition.
a = B*v
This 3-by-3 matrix B is called a skew-symmetric matrix.
