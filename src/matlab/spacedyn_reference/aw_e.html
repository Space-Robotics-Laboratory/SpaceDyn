aw

Grammar

E0 = aw( w0 )

global variable

d_time

Explanation

Function which calculates a coordinate transform matrix around an angular velocity vector w0 of the base.

Suppose the base rotates at a rate of w0.
If its attitude (coordinate transform matrix) at time t = n is IA0n ,
a linear approximation of its attitude at t = n + d_time is IA0n+d_time = aw( w0 ) * IA0n
