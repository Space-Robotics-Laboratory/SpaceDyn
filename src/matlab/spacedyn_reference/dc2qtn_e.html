dc2qtn

Grammar

qtn = dc2qtn( dc )

Global variables

---

Explanation

Function which calculates a quaternion from a direction cosine matrix.

Details

Consider a coordinate transform from I reference frame to J reference frame.
This transformation can be expressed as a rotation around some axis.
Let R be a vector of the axis of rotation, and th be an angle of rotation.
A quaternion is defined as follows.
qtn = [th]
